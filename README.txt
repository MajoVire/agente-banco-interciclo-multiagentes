# Sistema de Gestión Bancaria Inteligente

Sistema de simulación de agente inteligente para gestión de colas en agencias bancarias.

## Descripción
Sistema que implementa un agente inteligente para una agencia bancaria con gestión de colas, atención prioritaria y simulación en tiempo real.

## Características
- Gestión inteligente de colas
- Atención prioritaria (adultos mayores, embarazadas, etc.)
- 3 ventanillas de atención
- Interfaz gráfica profesional
- Simulación en tiempo real
- Sistema de logs detallado
